🔄 How The System Works

1

Slot Coordinates Loaded

slots.json contains 4 corner points for all 100 parking slots extracted from PKLot XML files.

2

Video Uploaded

User uploads a raw parking lot video through the dashboard. Flask saves it to static/uploads/.

3

Background Thread Starts

Flask creates a background thread to process video without freezing the browser.

4

Frame by Frame Processing

Each video frame is read and all 100 slot crops are extracted using OpenCV.

5

Batch CNN Prediction

All 100 crops sent to CNN in one batch. Model returns 100 predictions instantly — 10x faster!

6

Boxes Drawn

GREEN box = Empty slot. RED box = Occupied slot. Frame encoded as JPEG and stored.

7

Live Streaming

MJPEG streaming sends frames to browser in real time. Stats update every second.

8

Session Saved

All frame data saved to SQLite database. View complete history anytime in History page.
